git 开启区分大小写后, 删除重复的文件

  1. 让git区分大小写: git config core.ignorecase false
  2. 删除缓存: git rm -r --cached .
  3. 添加当前目录的所有文件: git add .
  4. 写注释: git commit -m 'fix: 修复git大小写不敏感造成的文件未上传'
  5. 提交: git push
